# MA30300 Optional Projects – Spring 2026

This repository contains **optional student projects** for the MA30300 course (Spring 2026).

## Overview
Students were given the freedom to choose **any project related to MA303 topics**. The goal was to explore mathematical concepts through programming, visualization, or computational tools.

## Example Project Ideas

Some directions included:

1. **Phase Portrait Tool**  
   Program a tool to plot the phase portrait of a system of differential equations.

2. **Laplace Transform Calculator**  
   Create a calculator that computes the Laplace transform and inverse Laplace transform of a function (it is acceptable to handle only certain types of functions).

3. **Numerical Methods for Differential Equations**  
   Implement methods such as Euler’s Method, Improved Euler, or Runge–Kutta to approximate solutions.  
   When possible, compare numerical results with exact solutions and analyze error.

## Expectations

- Demonstrate understanding of the **underlying mathematics**
- Ensure the project is **tested and functional**
- Clearly present results (e.g., demos or visualizations)
